# 主页

## 关于本站

本站旨在采用较新的网站技术整理马克思主义经典著作。

本站不是 [中文马克思主义文库](https://www.marxists.org/chinese/) 的镜像站，也不是 [马克思主义文库](https://www.marxists.org/) 的镜像站。本站与这两个站点没有任何关系。
